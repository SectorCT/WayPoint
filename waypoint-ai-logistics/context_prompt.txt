# WayPoint AI Logistics - Website Context Prompt

## Project Overview
WayPoint is a comprehensive AI-powered logistics management platform that revolutionizes delivery operations through intelligent automation, real-time tracking, and advanced route optimization. The platform consists of multiple interconnected applications serving different user roles in the logistics ecosystem.

## Website Structure & Technology Stack

### Frontend Architecture
- **Framework**: React 18.3.1 with TypeScript
- **Build Tool**: Vite 5.4.19
- **UI Library**: shadcn/ui components with Radix UI primitives
- **Styling**: Tailwind CSS 3.4.17 with custom animations
- **Routing**: React Router DOM 6.30.1
- **State Management**: TanStack React Query 5.83.0
- **Form Handling**: React Hook Form 7.61.1 with Zod validation

### Backend Integration
- **API**: Django REST Framework with PostgreSQL
- **Authentication**: JWT-based with role-based access control
- **Real-time**: WebSocket connections for live tracking
- **Routing Engine**: OSRM integration for optimal route calculation
- **Deployment**: Docker containerization
